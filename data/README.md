# Data folders

- `raw/`, `processed/`, `marts/`, `models/`, `reports/`: current VLR analytics pipeline datasets.
- `assets/`: local agent/team asset registries.
- `role_ranker/`: preserved role-ranker datasets from the Elo and tournament Streamlit projects.
- `legacy_2026_agents/`: preserved datasets from the original exploratory VLR agent scraper project.

The datasets are kept intentionally to avoid losing functionality or analysis state during the merge.
